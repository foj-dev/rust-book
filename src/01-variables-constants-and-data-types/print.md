# Print

The print! macro and its variations are used for debugging, informing the user, and generating formatted
output.

`print!`

```rust
// prints text without a line break
print!("Hello,");
print!("world!");
print!("...");
```

`println!`

```rust
// prints text with a line break
println!("Hello,");
println!("world!");
println!("...");
```
