# Constant

The constant is a variable with a fixed, immutable value, explicit type, and can be declared in any scope.

```rust
//
const PI: f32 = 3.14159;
println!("{}", PI);
```

```rust
//
const ERROR_404: &str = "Error 404!";
println!("{}", ERROR_404);
```
